# fake-news-detection-transformers-bilstm
Estudio comparativo de modelos Bi-LSTM, BERT y RoBERTa para la detección automática de desinformación sobre el corpus WELFake. TFG.
## Notebook experimental

El notebook principal del proyecto está disponible en:

[TFG_Alejandro_Fernández_Notebook.ipynb](./TFG_Alejandro_Fernández_Notebook.ipynb)

Este archivo contiene el flujo experimental completo: preparación del dataset WELFake, entrenamiento de Bi-LSTM, BERT y RoBERTa, evaluación de métricas y generación de figuras.
